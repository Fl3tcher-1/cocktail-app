about

I have been working at busy London bars and pubs since middle of 2019 and wish to compile a list of essential information in one place for all those who wish to enter or get better in the industry.
This coktailApp was built at the end of 2022/2023 using a free data source api giving the user multiple ways of searching for their new favourite cocktail to make at home.

using the search function


you can use the search bar to look for a specific cocktail, or you can also search by: whether a cocktail has a certain ingredient, is part of a certain category or is alcoholic


search by name

simply input a cocktail name and all matching ones will be returned to you


search by ingredient

this will allow you get cocktails that contain a certain ingredient. examples of search options you could use are:

*please be explicit for the ingredient you are searhcing for otherwise you may get unexpected results e.g. search for "blue curacao" instead of "blue" or "curacao" for those options are an incomplete ingredient*

  • spirit type:
  • gin
  • vodka
  • whiskey
  • rum
  • tequila

  • fruit:
  • banana

  • apple

    • search by category

    this refers to the category the drink is in--- cocktail, beer ordinary


    search by alcoholic

    currently you are not able to search for a specific non-alcoholic cocktail--- this will simply return all alcoholic or non alcoholic drinks
